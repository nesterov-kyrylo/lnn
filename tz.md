# PROMPT ДЛЯ ГЕНЕРАЦИИ ЛЕНДИНГА В VS CODE / AI CODE GENERATOR

Создай современный премиальный одностраничный сайт (landing page) для психолога.  
Сайт должен выглядеть дорого, спокойно, минималистично и вызывать доверие.  
Стиль — сочетание современной психологии, женственности, мягкости и экспертности.

## Основная цель сайта
Получение заявок на консультацию через форму записи.
